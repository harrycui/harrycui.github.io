Assignment

Description: Using Java to create a useful and challenging program! You can use whatever third-party tools, platforms, and IDEs. Just make sure you are not copying the code from others. Please fill the online form once you decide on your group member(s) and your project name.
Important Notes:
  • Each group should consist of 2 to 4 students
  • At least 5 user-defined classes
  • At least 1 UML diagram in your report
  • Runnable with Java 15
  • GUI (Graphical User Interface) is not essential
Assessment:
  • Program source code
  • Group presentation (8 minutes)
  • Group report (Template: U10M12004-OOP-Report-Group-XX.doc)
Submission: Each group should submit a group report (in PDF format) and a runnable Java package (or ZIP file) via email to Mr. Pengyu Liu (liupy8@mail.nwpu.edu.cn) before the last lecture (May 15). Make sure that your program is runnable with Java 15. The group presentations will be delivered in the last lecture.
